Time RttManager::GetFastRexmitTime ( ) const

Get the fast retransmission time.

The fast retransmission time is computed as follows:

FRXT = A + 4D

where A is the smoothed RTT and D is the smoothed mean deviation. This is for use when packets are lost and ACKs are not being delayed by a receiver.

Returns
The fast retransmission time.

Time RttManager::GetRexmitTime ( int  multiplier = 4) const

Get the retransmission time.

The retransmission time is computed as follows:

RXT = A + (M * D) + ACK_DELAY

where A is the smoothed RTT, M is the multiplier, D is the smoothed mean deviation, and ACK_DELAY is the amount of time that an ACK can be delayed by a receiver.

Parameters
multiplierThe smoothed mean deviation multiplier. Defaults to 4, per RFC 6298.
Returns
The retransmission time.

Time RttManager::GetRtoTime ( ) const

Get the RTO time.

The RTO time is computed as follows:

RTO = max( (A + 4D + ACK_DELAY), 200 msec )

where A is the smoothed RTT, D is the smoothed mean deviation, and ACK_DELAY is the amount of time that an ACK can be delayed by a receiver.

This is the standard RFC 6298 RTO time, with the addition of the ACK delay and the RTO floor changed from 1 second to 200 milliseconds to act more aggressively with retransmissions.

Returns
The next RTO time.
