virtual

Add a relationship between flows that means they are coupled. Coupled flows must all be admitted for any one of them to realize their utility.

NOTE: This method does not currently check or prevent adding a flow coupling multiple times. It also does not handle overlapping flow couplings.

Parameters

Returns
true on success, false otherwise.
