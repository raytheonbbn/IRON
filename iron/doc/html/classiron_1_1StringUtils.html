Detailed Description

A class that provides a set of static utility methods that deal with strings. This class enables us to capture frequently used string manipulation routines in a common place.

bool StringUtils::GetBool ( const std::string &  str,
const bool  default_value = true 
)
static

Convert the provided string to a boolean value.

The default value is used if there is an error interpreting the provided string value as a boolean.

Valid boolean values can be specified as:

  • Case insensitive characters 'true' evaluate to true
  • '1' evaluates to true
  • Case insensitive characters 'false' evaluate to false
  • '0' evaluates to false
Parameters
strThe string containing the value to be converted.
default_valueThe optional default boolean value to return if there is a conversion error. Defaults to true.
Returns
The boolean value for the provided string.
