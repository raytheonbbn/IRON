Detailed Description

This is the base class for process-specific stats like BpfStats. It contains the basic stats methods and members, like the method to set or access the dump interval.

iron::Time iron::Stats::last_dump_
protected

The last time the dump occurred. A dump may be triggered via the timer (every dump_interval_ms_) and via direct remote control requests. If the timer expires but a dump occurred less than dump_interval_ms_ - 10%, for instance via rc request, the dump is rescheduled for next timer expiration.
