Detailed Description

template<class C, typename O>
class iron::OrderedList< C, O >::WalkState

A class for maintaining state while walking the linked list.

An object of this class must be initialized using either the constructor or the PrepareForWalk() method in order to prepare for a walk of the linked list. Once the WalkState object is initialized, walk all of the links in order to examine each one. This is done by calling the GetNextItem() method repeatedly on the linked list with the same WalkState object.
