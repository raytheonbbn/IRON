115  // BPF Zombie packets. Note that this value is NOT stored at the start of

1410  // The following depicts a Packet buffer that is partially populated:
1411  //
1412  // |<------------------ kMaxPacketSizeBytes ------------------>|
1413  //
1414  // +---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+
1415  // | D | D | D | D | D | D | D | D | D | D | D | | | | |
1416  // +---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+
1417  // ^
1418  // |
1419  // start_
1420  //
1421  // |<---------------- length_ ---------------->|
1422  //
1423  // The bytes that have data in them are marked with a "D" in the figure,
1424  // the start of the packet is set to the first byte, and the length is
1425  // set to the bytes that have data in them.
1426  //
1427  // One method of modifying the Packet is to remove bytes from the
1428  // beginning of the buffer, accomplished via the
1429  // RemoveBytesFromBeginning() method. When this action is taken, the
1430  // buffer is modified as follows:
1431  //
1432  // |<------------------ kMaxPacketSizeBytes ------------------>|
1433  //
1434  // +---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+
1435  // | X | X | X | D | D | D | D | D | D | D | D | | | | |
1436  // +---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+
1437  // ^
1438  // |
1439  // start_
1440  //
1441  // |<---------- length_ ---------->|
1442  //
1443  // The start of the packet is moved forward by the number of bytes that
1444  // are removed (the bytes that are no longer part of the packet are marked
1445  // with an "X" in the figure) and the length of the packet is
1446  // appropriately adjusted.
1447  //
1448  // The Packet can also be modified by removing bytes from the end of the
1449  // buffer, accomplished via the RemoveBlockFromEnd() method. When this
1450  // action is taken, the buffer is modified as follows:
1451  //
1452  // |<------------------ kMaxPacketSizeBytes ------------------>|
1453  //
1454  // +---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+
1455  // | X | X | X | D | D | D | D | D | X | X | X | | | | |
1456  // +---+---+---+---+---+---+---+---+---+---+---+---+---+---+---+
1457  // ^
1458  // |
1459  // start_
1460  //
1461  // |<---- length_ ---->|
1462  //
