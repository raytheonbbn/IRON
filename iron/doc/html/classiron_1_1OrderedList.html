template<class C, typename O>
void iron::OrderedList< C, O >::Reposition ( WalkState &  ws,
const O &  new_value 
)
inline

Reposition a link in the linked list.

This is to be used if the order of an item in an ordered List changes. It removes the link and reinserts it in the correct position to restore ordering in the list. This only works is there is no more than one item that is out of order and should therefore be called immediately after changing the order of an item in an ordered list.

Warning: After this call, the walkstate will be pointing to the item before the old position of item being repositioned, thus continuing to use the walkstate can lead to items being skipped or repeated.

Parameters
wsA reference the walkstate which points to the item being repositioned.
new_valueThe value by which to order this element in the list.
