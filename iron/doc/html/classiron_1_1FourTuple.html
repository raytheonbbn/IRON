Detailed Description

A class for an IPv4 four-tuple that uniquely identifies a flow.

Contains a source IPv4 address, a source TCP/UDP port number, a destination IPv4 address, and a destination TCP/UDP port number. The addresses and ports are stored in Network Byte Order.

Note that this class is not designed to be extended (inherited from) in order to maintain efficiency. Specifically, the destructor is not virtual in order to eliminate the VTABLE.
