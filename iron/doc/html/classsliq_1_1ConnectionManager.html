Add a new connection object.

The connection manager takes ownership of the object. Any connection with the same endpoint ID is destroyed.

bool ConnectionManager::DeleteConnection ( EndptId  endpt_id)

Schedule a connection for deletion.

The connection object, if found, is scheduled to be destroyed at a later time. This makes it safe to be called from the connection object that needs to be destroyed.

Parameters
endpt_idThe endpoint ID of the connection to be destroyed.
Returns
True if the connection object was found and scheduled for destruction, or false otherwise.
