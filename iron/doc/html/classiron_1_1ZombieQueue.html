Detailed Description

A class that will maintain the number of zombie bytes enqueued for a destination. bytes. Zombie packets are created during Dequeue.

Generates and returns a Zombie Packet of up to the specified size.

If there are no enqueued zombie bytes, then NULL is returned as the object. If a packet is generated/returned, the caller takes ownership of the memory.

Parameters

dst_vecDestinations we'd like to dequeue for. MUST be 0 unless this is a packetless zombie queue.
Returns

bool ZombieQueue::Enqueue ( Packet *  pkt)
virtual

Enqueue an element into the queue.

This places the elements at the tail end of the queue. Once an object is enqueued, the queue takes ownership of the memory.

If the queue is already full when this method is called, then an existing packet in the queue is attempted to be dropped based on the current drop policy. Any dropped packet is silently deleted.
