Detailed Description

The ASAP class contains the state and logic for the ASAP algorithm.

ASAP (Anti-Starvation with Artificial Packets) proactively adds zombies for a group,destination that is being starved due to insufficient packets entering the node.

void ASAP::AdjustQueueValuesForAntiStarvation ( )

Adjust the queue depths for anti-starvation.

This will add zombies to the queue to account for packet delay, with the goal of preventing starvation by adding more and more zombies as the delay grows.
