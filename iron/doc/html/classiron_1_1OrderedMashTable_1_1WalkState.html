Detailed Description

template<class K, class V, typename O>
class iron::OrderedMashTable< K, V, O >::WalkState

A class for maintaining state while walking a mash table.

An object of this class must be initialized using either the constructor or the PrepareForWalk() method in order to prepare for a walk of the mash table. This is to be used only for iterating the mash table and the the mash table should not be modified while walking. Items should be removed based on key, rather that using the WalkState as a reference point.
