Detailed Description

Class for storing Packet references in the shared memory packet pool.

The implementations ShmPPCircBuf and LocalPPCircBuf are almost identical. The Get() and Put() methods are exactly identical. However, these must still be separate classes because (a) we need statically allocated arrays so we can use these in shared memory, and (b) we can't use inheritence to achieve code reuse without having virtual functions, and virtual functions would result in a vtable that throws off our shared memory allocation and pointer manipulation.

This class is public so it can be tested in packet_pool_test.cc.
