Detailed Description

template<class C>
class iron::BinIndexableArrayNc< C >

Templated Bin Indexable Array No-Copy class.

Used for classes ("C" below) that do not have a copy operator. This template class is similar to BinIndexableArray, but does not have a Clear() method.

Initialize by calling Initialize() passing the BinMap object.
