Detailed Description

template<class C>
class iron::BinIndexableArrayShm< C >

Templated Bin Indexable Array Shared Memory class.

Initialize by:

  • calling Initialize() passing the BinMap object,
  • setting up the SharedMemoryIF object using the size returned by GetMemorySizeInBytes(), and
  • calling SetShmDirectAccess() passing the SharedMemoryIF object.

Note that the shared memory must be locked and unlocked as necessary outside of this class when using Clear() or operator[]().
