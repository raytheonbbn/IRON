Detailed Description

template<class C>
class iron::List< C >

The templated non-ordered Linked List class.

The linked list is not ordered. Items should be added using the Push(), and pulled out using Peek() / Pop() for FIFO operation. Use Push(), Peek() and PopBack() for LIFO operation.

The item class destructors cannot be relied upon to manage external memory. This template creates internal storage objects and manages the memory used by these internal storage objects. Any necessary memory management of the items stored in the List must be handled by the user of this template.
