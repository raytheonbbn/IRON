Detailed Description

Manages the SLIQ sockets.

Currently, this class is capable of managing up to FD_SETSIZE sockets, which should be sufficient for virtually all applications. The recvmmsg() system call is used for reading from the sockets, which is capable of receiving multiple packets for each system call.
