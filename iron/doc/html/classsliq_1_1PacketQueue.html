Packet * PacketQueue::Dequeue ( )

Remove a packet from the queue.

This method is non-blocking. The packet is removed from the head of the queue if the dequeueing rule is FIFO_QUEUE, or the tail of the queue if the dequeueing rule is LIFO_QUEUE. If there are no packets in the queue, then NULL is returned. If a packet is dequeued, the caller takes ownership of the memory.

Returns
Returns a pointer to the packet being dequeued, or NULL if there are no packets to dequeue.
