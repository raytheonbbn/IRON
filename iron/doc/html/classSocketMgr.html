void SocketMgr::RemoveAllSockets ( )

Remove all Sockets.

This is typically called when the process is terminating. This enables us to delete the dynamically allocated Sockets and not rely on the order that the static Singleton SocketMgr is deleted. We do this because the Sockets have timers, which should be cleared from the static Singleton TimerMgr.
